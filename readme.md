# UntisAPI
## Description
This is a wrapper for the untis api in go. 
Currently, it is very limited and supports only authentication using Username/Secret.

### Supported methods:
- Login
- GetClasses
- GetLatestImportTime
- GetSchoolyears
- GetLatestSchoolyear

## Installation
```
go get github.com/tomroth04/untisApi
```
